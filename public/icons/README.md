# PWA Icons

This directory should contain PWA icons for the application.

## Required Icons

- `icon-192.png` - 192x192 pixels (for Android home screen)
- `icon-512.png` - 512x512 pixels (for splash screen and high-resolution displays)

## Icon Generation

Icons can be generated using:
- Online tools (e.g., PWA Asset Generator)
- Image editing software (Photoshop, GIMP, etc.)
- Command-line tools

## Design Guidelines

- Use a simple, recognizable design
- Ensure good contrast for visibility
- Follow platform-specific guidelines (iOS, Android)
- Test on actual devices

## Note

For development, placeholder icons can be used. For production, proper icons should be created and added to this directory.
